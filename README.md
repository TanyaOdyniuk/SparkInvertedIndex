Построить инвертированный индекс для датасета представляющего собой набор сообщений.
Данные представляют собой файлы с текстом сообщений разложенные по каталогам (20_newsgroup). Имя файла - это индекс записи.

В результате должен получиться файл index.csv содержащий три поля для каждого слова: слово, общее количество вхождений этого слова (по всем документам), список документов через пробел. 

Например:

  accept,258,54243 60890 60884 76067 105252
  crud, 4, 104959 16152
(результат приведен только для примера)

При считывании файлов следует:

Исключить строки со служебной информацией: Path:, Newsgroups:, xxx writes:, ...;

Запятые, точки, пробелы, etc. считать разделителями;

Брать только слова, состоящие из латинсих букв и символа ';

Все слова привести к нижнему регистру;
